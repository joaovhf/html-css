Estilo Interno

Tag

Para utilizar o CSS Interno, abre-se a tag "style" no cabeçalho(head) do código HTML.

Parâmetros

Dentro da tag "style" você vai criar seletores (reptir o nome da tag original) referentes às tags que você quer modificar. Aí você faz isso digitando o nome da tag e abrindo chaves ({)

Adicionar estilos

Dentro de cada seletor que você abrir, vai colocar os parâmetros com os estilos que você quer aplicar à tag original

Vantagem em relação ao Inline

A vantagem do CSS Interno é que você pode aplicar um estilo a diversos elementos de uma mesma tag. Já no Inline, você faz individualmente para cada elemento que você quer modificar.

Desvantagens

  • A parte de estilo acaba ocupando muitas linhas do seu código e você às vezes tem que rolar muito pra baixo para chegar no body.

  • O parâmetro style só serve para a página que você está aplicando. Se o seu site tiver muitas páginas terá que copiar e colar o código em todas e, caso tenha que fazer modificações, terá que mudar em cada uma página.
